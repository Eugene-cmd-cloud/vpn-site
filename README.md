# GitHub Pages VPN mini-site

## What is inside
- `index.html` — landing page in Liberty-like style
- `subs/vpnd-moldova-liberty.json` — JSON config for Liberty/Xray-style import
- `subs/vpnd-moldova.txt` — plain text subscription with one VLESS link

## How to publish
1. Create a GitHub repository.
2. Upload all files from this folder.
3. In GitHub: **Settings -> Pages**.
4. Source: **Deploy from a branch**.
5. Branch: **main**, folder: **/(root)**.
6. Save.

## Example links after publish
If your Pages address is:
`https://YOURNAME.github.io/YOURREPO/`

Then the files will be:
- Site: `https://YOURNAME.github.io/YOURREPO/`
- JSON: `https://YOURNAME.github.io/YOURREPO/subs/vpnd-moldova-liberty.json`
- TXT subscription: `https://YOURNAME.github.io/YOURREPO/subs/vpnd-moldova.txt`

## Notes
- The page copies absolute links automatically after deployment.
- Replace texts, support link, and branding inside `index.html` if needed.
